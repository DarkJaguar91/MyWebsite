My third year Computer Science Project.

A visualiser and comparison tool for the UCT biology department's element reduction height maps.

Displays the height maps as heat maps. The user can draw a line on the heat map to draw a line graph of the values along that line. Can compare 2 height maps via a difference map, graph, 3D drawer and RMSD values.

Developed in Java and OpenGL.
