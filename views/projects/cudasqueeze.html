Honours Project

UCT Computer Science

SKA South Africa requires a compression system for their radio telescop data coming on from the satelite dishes. This compression system is required to be done in "Real Time", this means that the data has to be compressed at the same rate it could receive the data.

We decided that in order to find the best algorithm, one which could compress at the required speed and give a good compression ratio, we would have each group memeber create a type of algorithm, both for a multi-threaded CPU system and a GPU system using CUDA.
