Computer Games Design 3rd year project

UCT Computer Science

Grade: 100%

A 3D game based on the 1996 Future Cop: LAPD playstation 1 game.

The task was to create a 3D XBOX 360 game using the Microsoft XNA API. NEBULON 12, first called BBN-GAME since it was an acronym for all the team members names, was the only game submitted that year to have full 3D movement like a airplane, and the only game to have a working "2D" representation of what is currently happening in the game (an interactive map).
